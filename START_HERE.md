# Start Here

This repository is the control room for Algorzm's AI Operating System.

Use this file as the first stop when opening the GitHub repository or starting a new AI worker shift.

## Fast Path

1. Read `CONSTITUTION.md` for the mission and principles.
2. Read `ARCHITECTURE.md` for the operating-system hierarchy.
3. Read `AGENTS.md` for runtime behavior.
4. Read `.handoff/latest.yml` for the latest shift state.
5. Read `portfolio/project_cells/README.md` to find the relevant project.
6. Continue from that project's `PROJECT_CELL.md`.

## Human Navigation

| Need | Go To |
| --- | --- |
| Understand the whole system | `REPOSITORY_MAP.md` |
| See all active projects | `portfolio/project_cells/README.md` |
| Continue interrupted work | `.handoff/latest.yml` |
| Route work with minimal tokens | `mission-control/SUSTAINABLE_DEVELOPMENT.md` |
| Run or improve the AI OS code | `alos/` |
| Review operating rules | `governance/`, `mission-control/`, `AGENTS.md` |
| Check generated outputs | `outputs/` |
| Inspect the dashboard prototype | `site/` |

## Operating Rule

Do not rely on chat history as memory.

If a decision, status, blocker, or next action matters, record it in the relevant Project Cell, handoff file, issue, or operating-system document.
