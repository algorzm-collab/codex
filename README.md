# Algorzm AI Operating System

This repository is the shared brain for Algorzm's AI agents.

It is not only a Codex configuration repository.
It is a continuously evolving operating system for AI-assisted problem solving, consulting, product building, and software engineering.

## Purpose

The goal is to make every AI agent work with Algorzm's problem-solving philosophy.

AI agents should not merely produce code.
They should:

1. Understand the real problem
2. Think strategically
3. Research before building
4. Use tools and libraries aggressively
5. Avoid shallow vibe coding
6. Preserve user feedback as reusable memory
7. Improve the system after every meaningful task

## Core Idea

Every interaction should create reusable intelligence.

Feedback becomes rules.
Rules become playbooks.
Playbooks become skills.
Skills become better execution.

## Repository Structure

- `AGENTS.md`  
  Main instruction file for AI agents.

- `brain/feedback/`  
  Raw user feedback and daily learning logs.

- `brain/rules/`  
  Reusable operating principles.

- `brain/playbooks/`  
  Step-by-step workflows for recurring tasks.

- `brain/memory/`  
  Long-term preferences, mistakes, learnings, and patterns.

- `brain/decisions/`  
  Important decisions and tradeoffs.

## Human Role

The human gives judgment, direction, feedback, and taste.

## AI Role

The AI agent executes, reasons, records, improves, and updates the shared brain.

## Golden Rule

Do not leave important knowledge trapped inside chat history.

If something was learned, store it.
