# STRAT Case Study Template v1

## 사용 목적

홈페이지의 사례는 실적 목록이 아니라 신뢰를 만드는 증거다.  
각 사례는 다음 구조로 정리한다.

```text
고객 유형 -> 문제 -> STRAT 접근 -> 산출물 -> 효과 -> 상담 CTA
```

## 사례 선정 기준

대표 사례 6개는 아래 기준으로 고른다.

1. STRAT의 주력 분야를 보여주는가
2. 방문자가 자기 문제로 느낄 수 있는가
3. 산출물이 명확한가
4. 유사 고객에게 상담 이유를 만드는가
5. 공개 가능성이 있는가
6. 7대 솔루션 클러스터 중 하나 이상과 연결되는가
