# strategy 홈페이지 고도화를 위한 자료 요청 패킷 v1

## 1. 요청 목적

홈페이지를 더 강하게 만들기 위해 필요한 자료는 단순 회사소개서가 아니다.  
방문자가 “이 회사가 내 문제를 해결해본 적이 있구나”라고 느끼게 만드는 증거 자료가 필요하다.

자료는 아래 6개 영역으로 받는다.

```text
회사 정체성
-> 대표/구성원 신뢰
-> 방법론
-> 실적
-> 핵심장표
-> 문의 전환 콘텐츠
```

## 2. 회사소개 자료

필요 자료:

- 공식 회사명 표기
- 브랜드 표기: strategy / STRATEGY / strat.kr 중 어떤 것을 쓸지
- 회사 한 줄 소개
- 회사 3문장 소개
- 회사 상세 소개
- 설립연도
- 대표 전화
- 대표 이메일
- 주소
- 사업자/법인 정보 공개 여부
- 로고 원본
- 기존 회사소개서
- 기존 제안서 표지

홈페이지에서 쓸 목적:

- 회사소개 섹션
- Footer
- 관리자 기본 설정
- 문의 영역

## 3. 김정성 대표/구성원 소개 자료

필요 자료:

- 김정성 대표 프로필
- 학력/경력/주요 이력
- 대표 전문 분야
- 대표 수행 프로젝트 10개
- 대표 사진 사용 여부
- “Sheer Excellence” 문구 사용 여부
- Sein & Sollen 철학 설명
- Fermat Point Methodology에 대한 직접 설명
- 연구본부/전문위원 소개
- 구성원별 이름/직함/전문 분야/사진 공개 여부

홈페이지에서 쓸 목적:

- People 섹션
- 대표 프로필 페이지
- 방법론 신뢰 강화
- 문의 전환 전 신뢰 레이어

## 4. 방법론 자료

필요 자료:

- 1st Advisory 정의
- Sein & Sollen 설명
- Fermat Point Methodology 설명

- 문제해결 프로세스

- 전략수립 방법론
- 조직진단/조직재설계 방법론
- 성과관리/KPI 방법론
- 인사제도/직무분석 방법론
- ESG/사회적가치 방법론
- AI/DT 전략 방법론

가능한 파일:

- PPT
- PDF
- 이미지
- 도식
- 제안서 일부
- 보고서 목차

홈페이지에서 쓸 목적:

- Method 섹션

- Issue Finder 결과 화면
- 서비스 상세 페이지

## 5. 회사 실적 자료

현재 받은 실적은 DB에 65개 반영했다.  
이제 필요한 것은 “홈페이지에 보여줄 대표 실적”과 “전체 실적 관리용 자료”를 분리하는 것이다.

### 5.1 전체 실적

필요 컬럼:

- 테마
- 산업
- 고객명
- 프로젝트명
- 연도
- 공개 가능 여부
- 비고

현재 테마:

- Strategy 전략수립
- Organization & Personnel 조직재설계
- Performance Management 성과관리
- HR System 인사관리

추가 가능 테마:

- ESG/사회적가치
- AI/DT Transformation
- 경영평가 대응
- 사업타당성/시장조사
- 교육/워크숍

### 5.2 대표 실적 12개

홈페이지 메인에 올릴 대표 실적이 필요하다.

추천 구성:

1. 전략수립 대표 실적 3개
2. 조직재설계 대표 실적 3개
3. 성과관리 대표 실적 2개
4. 인사관리 대표 실적 2개
5. ESG/AI/신사업 대표 실적 2개

각 대표 실적마다 필요한 내용:

```text
고객명:
공개 가능 여부:
산업:
문제:
strategy 접근:
산출물:
성과/효과:
홈페이지 노출 가능 문장:
```

홈페이지에서 쓸 목적:

- 관련 실적 섹션
- Issue Finder 결과별 자동 추천
- 서비스 상세 페이지
- 영업 신뢰 근거

## 6. 핵심장표 자료

홈페이지 퀄리티를 결정하는 핵심은 장표다.  
좋은 컨설팅 홈페이지는 “말”보다 “정리된 사고의 화면”을 보여줘야 한다.

필요 장표:

- 전략체계도
- Sein & Sollen 도식
- Fermat Point Methodology 도식
- 전략-KPI 연결 맵
- R&R 매트릭스
- 조직진단 프레임
- 성과관리 운영체계
- KPI 설계 예시
- 직무분류체계 예시
- 역량모델 예시
- 중장기 로드맵 예시
- 프로젝트 진행 프로세스

자료 형태:

- 원본 PPT
- 이미지 PNG
- PDF 캡처
- 민감정보 제거본
- 흐림 처리본

주의:

- 고객명/수치/내부정보는 삭제해도 된다.
- 홈페이지에는 “진짜 컨설팅 산출물의 질감”만 보여주면 된다.

## 7. Issue Finder 고도화 자료

현재 Issue Finder는 4개 결과 유형이 있다.

- 전략-조직-성과 연결부 약화
- R&R 및 조직역할 혼선
- KPI/성과관리 작동 불량
- 전환과제 실행화 미흡

각 결과 유형별로 필요한 자료:

```text
문제 이름:
고객이 자주 하는 말:
이 문제가 생기는 원인:
초기 상담에서 확인할 질문:
관련 대표 실적:
추천 가설 솔루션:
예상 산출물:
문의 CTA:
```

홈페이지에서 쓸 목적:

- 결과 화면 고도화
- 관련 실적 추천
- 상담 전환율 개선

## 8. 인사이트/콘텐츠 자료

SEO와 신뢰 형성을 위해 아래 콘텐츠가 필요하다.

우선 작성할 글:

- 보고서는 있는데 실행이 안 되는 이유
- 전략과 KPI가 따로 움직이는 이유
- 조직개편보다 R&R 재정립이 먼저 필요한 경우
- 공공기관 중장기전략 수립 전 체크리스트
- 성과평가제도가 성과를 만들지 못하는 이유
- Sein & Sollen으로 보는 전략 컨설팅
- Fermat Point Methodology란 무엇인가

각 글에 필요한 것:

- 핵심 메시지
- 실제 프로젝트에서 자주 본 문제
- 고객이 조심해야 할 점
- strategy의 관점
- 관련 서비스/문의 CTA

## 9. 문의/영업 전환 자료

필요 결정:

- 문의 받을 이메일
- 문의 받을 전화번호
- 상담 가능 시간
- 1차 상담 무료 여부
- 문의 후 응답 기준 시간
- 상담 방식: 전화/화상/방문
- 고객에게 처음 요청할 정보

권장 문의 문구:

> 홈페이지 결과는 간단한 선택을 바탕으로 한 가설입니다. 실제 과제 범위와 접근 방식은 strategy와의 상담에서 구체화합니다.

## 10. 우선순위

가장 먼저 필요한 자료:

1. 대표 실적 12개 상세 설명
2. Fermat Point Methodology 원본 장표
3. Sein & Sollen 설명 자료
4. 김정성 대표 프로필/사진
5. 전략체계도/KPI/R&R 등 핵심장표 샘플
6. 문의 이메일/전화번호 확정

## 11. 보내주면 바로 할 일

자료를 받으면 바로 다음 작업을 진행한다.

1. 대표 실적을 홈페이지 카드로 변환
2. Issue Finder 결과별 관련 실적 매칭 개선
3. 김정성 대표 프로필 섹션 고도화
4. Method 섹션을 실제 장표 기반으로 재구성
5. 핵심장표 갤러리 고도화
6. 문의 전환 문구 개선
7. 관리자 데이터에 실적/장표 반영
