## Recovery Protocol

At the start of a new chat or agent shift:

1. Read `CONSTITUTION.md`, `ARCHITECTURE.md`, and `AGENTS.md`.
2. Read `.handoff/latest.yml` if it exists.
3. Read this file.
4. Read the relevant `PROJECT_CELL.md`.
5. Continue from `Next Action` without asking the CEO to reconstruct context.
