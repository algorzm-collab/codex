# Operation Commands

## Portfolio Brief

CEO command:

```text
포트폴리오 브리핑
```

Expected response:

```text
ALGORZM Portfolio Cockpit

프로젝트 | 상태 | 진행률 | 다음 액션 | CEO 필요 | Issue
---|---|---:|---|---|---
...
```

Source:

- GitHub issues with `project-cell` label

---

## Project Update

CEO command:

```text
<프로젝트명> Project Cell 업데이트
```

The CTO updates the related GitHub issue comment or body.

---

## Micro Task Creation

CEO command:

```text
<프로젝트명> Codex micro task 만들어
```

The CTO creates a separate micro-task issue linked to the Project Cell.

---

## Rule

Project Cell issue is the long-term room.

Micro-task issue is the execution unit.

ChatGPT is the renderer and CTO.
