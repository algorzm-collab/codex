# 홈페이지 v2 마스킹 검토 목록

> **공개 배포 금지 — CEO 승인 대기.** 아래 표기는 정보가치 보존형 마스킹의 검수안이다. 이 목록 전체가 승인되기 전에는 배포, 업로드, 공개 링크 생성을 진행하지 않는다.

## 마스킹 규칙

- 공개: 기관유형, 수행연도, 과제 성격, 문제, 접근, 결과물
- 비공개: 고객 실명, 이전 회사명, 계약금액, 실명을 추정하게 하는 약칭
- 표기 형식: `기관유형(YYYY)`
- 연도 원칙: 원천 수행기록의 과제명에 명시된 연도만 사용한다. 목표연도(예: 2030·2035)는 수행연도로 사용하지 않는다.
- 증빙 원칙: 회사 수행경험과 대표 컨설턴트 총괄 수행경험을 섞어 쓰지 않는다.

## UI 점검 범위

- Hero 및 메뉴: 고객 실명 없음
- 수행경험 지도: 위 9개 표기만 사례 이름으로 사용
- 상세 패널: 고객 실명·이전 회사명 없음
- 사례 심층: 위 9개 중 3개 표기만 사용
- 사람·문의: 고객 실명·이전 회사명 없음
- 검색: 입력값과 일치하는 이미 공개된 마스킹 표기만 강조
